Termo de Privacidade

Termo de Privacidade do Aplicativo OnLab

1. Informações Gerais

Este Termo de Privacidade descreve como o aplicativo On Lab, uma plataforma independente para gerenciamento de reservas de laboratórios, coleta, utiliza e protege as informações dos usuários.

2. Informações Coletadas

O On Lab coleta os seguintes dados para funcionamento adequado do sistema:

  • Dados de identificação: E-mail utilizado para login.
  • Dados de acesso: Senha criada pelo usuário para autenticação no aplicativo.
  • Dados de uso: Informações sobre as reservas realizadas, incluindo data, horário, laboratório e finalidade da reserva.

3. Finalidade da Coleta de Dados

Os dados coletados são utilizados para:

  • Gerenciamento de reservas: Permitir que os usuários realizem e acompanhem suas reservas.
  • Controle de acesso: Garantir que apenas usuários autorizados utilizem o sistema.
  • Melhoria da plataforma: Analisar o uso para aprimorar a experiência do usuário.
  • Notificações: Enviar atualizações sobre reservas e mudanças no sistema.

4. Compartilhamento de Dados

O On Lab não compartilha dados com terceiros, exceto quando exigido por lei ou determinação judicial.

5. Segurança dos Dados

Empregamos medidas de segurança para proteger as informações contra acessos não autorizados. As senhas são criptografadas e armazenadas de forma segura.

6. Armazenamento de Dados

Os dados dos usuários são armazenados em servidores protegidos e mantidos pelo tempo necessário para cumprir as finalidades descritas neste termo.

7. Direitos dos Usuários

Os usuários podem solicitar a alteração ou exclusão de seus dados a qualquer momento. Para isso, basta entrar em contato pelo e-mail adm.etedaf@gmail.com.

8. Alterações a este Termo de Privacidade

Este termo pode ser atualizado periodicamente. O usuário deve revisá-lo regularmente para se manter informado sobre eventuais mudanças.

9. Aceitação deste Termo de Privacidade

Ao utilizar o On Lab, o usuário concorda com os termos aqui descritos. Caso não concorde, o uso do aplicativo deve ser interrompido.

10. Contato

Para dúvidas ou solicitações sobre este termo, entre em contato pelo e-mail adm.etedaf@gmail.com.
