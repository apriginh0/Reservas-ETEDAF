Termo de Privacidade

Termo de Privacidade do Aplicativo OnLab

1. Informações Gerais

Este Termo de Privacidade descreve como o aplicativo OnLab, desenvolvido para uso exclusivo de professores da Escola Técnica Estadual Deputado Afonso Ferraz, coleta, utiliza e protege as informações dos usuários.

2. Informações Coletadas

O aplicativo OnLab coleta as seguintes informações dos usuários:

  • Dados de identificação: E-mail institucional do usuário.
  • Dados de acesso: Senha criada pelo usuário para acesso ao aplicativo.
  • Dados de uso: Informações sobre as reservas de laboratório realizadas, como data, horário, laboratório selecionado e finalidade da reserva.

3. Finalidade da Coleta de Dados

As informações coletadas pelo aplicativo OnLab são utilizadas para as seguintes finalidades:

  • Gerenciamento de reservas: Permitir que os professores realizem e gerenciem reservas de laboratórios de forma eficiente.
  • Controle de acesso: Garantir que apenas professores autorizados tenham acesso aos laboratórios.
  • Melhoria do aplicativo: Analisar dados de uso para identificar problemas e melhorar a funcionalidade do aplicativo.
  • Comunicação: Enviar notificações sobre reservas, alterações e outras informações relevantes.

4. Compartilhamento de Dados

O aplicativo OnLab não compartilha dados dos usuários com terceiros, exceto nos casos em que for exigido por lei ou por decisão judicial.

5. Segurança dos Dados

O aplicativo OnLab utiliza medidas de segurança avançadas para proteger os dados dos usuários contra acesso não autorizado, uso indevido ou alteração. As senhas dos usuários são criptografadas e não são armazenadas de forma legível no banco de dados.

6. Armazenamento de Dados

Os dados dos usuários são armazenados em servidores seguros, localizados em data centers com alta disponibilidade e segurança. Os dados são mantidos pelo tempo necessário para cumprir as finalidades descritas neste Termo de Privacidade.

7. Direitos dos Usuários

Os usuários têm o direito de acessar, corrigir ou excluir seus dados pessoais a qualquer momento. Para exercer esses direitos, o usuário deve entrar em contato com a administração da Escola Técnica Estadual Deputado Afonso Ferraz.

8. Alterações a este Termo de Privacidade

Este Termo de Privacidade pode ser alterado a qualquer momento, sem aviso prévio. É responsabilidade do usuário verificar periodicamente este Termo de Privacidade para estar ciente de quaisquer alterações.

9. Aceitação deste Termo de Privacidade

Ao utilizar o aplicativo OnLab, o usuário concorda com os termos deste Termo de Privacidade. Caso não concorde com estes termos, o usuário não deve utilizar o aplicativo.

10. Contato

Em caso de dúvidas sobre este Termo de Privacidade, entre em contato com a administração da Escola Técnica Estadual Deputado Afonso Ferraz.
